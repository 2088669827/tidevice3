# tidevice3
wrapper for pymobiledevice3 to make it more easy to use.


# Install

```bash
pip install tidevice3

# or install as Isolated environment
brew install pipx
pipx install tidevice3
```

# Usage
```bash
$ t3 list
...

$ t3 developer
# enable developer mode and mount develoepr image
```
